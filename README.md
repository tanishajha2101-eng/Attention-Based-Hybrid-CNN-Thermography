 Attention-Based Hybrid CNN for Breast Cancer Detection using Thermography

This repository contains the implementation of an attention-based hybrid CNN model for breast cancer classification using thermographic images.
 Dataset
BCD Thermography Dataset (private). Images are not shared due to ethical restrictions.

 Methodology
- Handcrafted features + SVM
- Deep feature extraction
- End-to-end CNN
- Attention-based hybrid CNN (Proposed)

 Results
- Accuracy: 99.02%
- AUC: 0.9981

Visualization
-Learning Curve
- Confusion Matrix
- ROC Curve
- Precision–Recall Curve
- Attention Heatmaps (Grad-CAM)

 Experimental Results

The following figures summarize the performance of the proposed Attention-Based Hybrid CNN model:

- Learning Curve: Training and validation accuracy and loss across epochs.
- Confusion Matrix: Classification results on the test dataset.
- ROC Curve: Receiver Operating Characteristic curve showing discriminative ability.
- Precision–Recall Curve: Precision vs recall trade-off.
- Attention Heatmap: Visualization of discriminative thermal regions contributing to classification.

 Note: Patient images are not shared due to ethical and privacy constraints.
